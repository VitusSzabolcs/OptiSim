void bind_optical_objects ( py::module_ &  m)

Binds the C++ OpticalObject class and its derived classes to Python.

This function defines the Python interface for OpticalObject, Lens, ThinLens, and ThickLens classes, making their constructors and methods accessible from Python.

Parameters
mA reference to the pybind11 module to which the classes will be bound.

Python binding for the OpticalObject base class.

Represents an abstract optical component with a position along the optical axis.

Python binding for the Lens base class, derived from OpticalObject.

Represents an abstract lens with a focal length.

Python binding for the ThinLens class, derived from Lens.

Represents an idealized thin lens with a single principal plane.

Python binding for the ThickLens class, derived from Lens.

Represents a real-world thick lens with non-negligible thickness and curved surfaces.
