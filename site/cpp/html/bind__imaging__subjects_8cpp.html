void bind_imaging_subjects ( py::module_ &  m)

Binds the C++ ImagingSubject class and its derived classes to Python.

This function defines the Python interface for ImagingSubject, LightSource, and Image classes, making their constructors and methods accessible from Python.

Parameters
mA reference to the pybind11 module to which the classes will be bound.

Python binding for the ImagingSubject base class.

Represents an abstract subject within the imaging simulation, possessing X and Y coordinates.

Python binding for the LightSource class, derived from ImagingSubject.

Represents a light-emitting source within the imaging simulation.

Python binding for the Image class, derived from ImagingSubject.

Represents an image formed by light, which can be real or virtual.
