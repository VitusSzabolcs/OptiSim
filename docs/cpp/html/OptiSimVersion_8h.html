Retrieves the full version string of the OptiSim library.

Returns
A constant C-style string containing the OptiSim library version (e.g., "1.0.0").

This function provides direct access to the OPTISIM_VERSION_STRING macro, which defines the current version of the OptiSim library in "MAJOR.MINOR.PATCH" format.
